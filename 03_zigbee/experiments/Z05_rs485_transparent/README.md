# Z5 Zigbee-RS485无线透传与工业数据采集实验

## 基本信息

| 项目 | 内容 |
| --- | --- |
| 方向 | Zigbee |
| 类型 | 进阶 |
| 难度 | 中 |
| 主要设备 | ZigBee 3.0转RS485模块、串口/Modbus模拟器 |
| 协议/接口 | Zigbee/RS485/Modbus可选 |
| 负责人 | 兰佳和 |
| 协作人 | 公共测试负责人 |
| 成果物 | 代码+指导书+PPT |

## 实验目标

完成RS485数据透传、轮询采集、异常帧处理，适配工业数据采集场景

## 备注

可用串口模拟器替代真实传感器

## 目录结构

```text
Z05_rs485_transparent/
  src/      实验源码
  config/   配置文件
  docs/     实验指导书、说明文档
  slides/   PPT 或汇报材料
  tests/    测试脚本、测试记录
  assets/   接线图、截图、素材
```

## 每周任务

| 周次 | 工作重点 | 主要任务 | 本周交付物 | 验收标准 |
| --- | --- | --- | --- | --- |
| 第1周 | Zigbee工业/RS485 | 熟悉Zigbee-RS485透传模块；确认串口参数、接线方式、Modbus/模拟串口数据源；设计工业采集综合实验流程。 | Z5/Z8实验大纲、RS485接线图、串口参数表、风险点清单。 | 完成RS485/串口透传最小闭环或模拟闭环验证。 |
| 第2周 | 实现RS485透传基础 | 完成Zigbee-RS485模块接线、串口参数设置、透传测试；编写工业采集脚本，可接入Modbus模拟器或串口模拟数据。 | Z5代码、接线图、串口参数表、指导书初稿、PPT提纲。 | RS485数据可通过Zigbee无线链路到达PC端并被解析。 |
| 第3周 | 完成工业采集综合场景 | 把Z5 RS485透传接入Z8工业采集流程；对接MQTT/VSOA桥接；补充异常帧、断连恢复、排错说明。 | Z5二版、Z8综合代码、指导书初稿、PPT主体、测试记录。 | Z8能完成RS485数据采集→MQTT→VSOA查询的闭环。 |
| 第4周 | Zigbee工业课程定稿 | 完成Z5/Z8最终材料；补充RS485排错、异常处理、工业场景说明；准备综合演示。 | Z5/Z8最终代码、指导书定稿、PPT定稿、测试记录。 | Z8可完成端到端工业采集演示，排错说明完整。 |
