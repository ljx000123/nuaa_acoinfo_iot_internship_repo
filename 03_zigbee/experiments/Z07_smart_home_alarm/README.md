# Z7 智能家居环境监测与联动告警实验

## 基本信息

| 项目 | 内容 |
| --- | --- |
| 方向 | Zigbee |
| 类型 | 综合 |
| 难度 | 高 |
| 主要设备 | CC2530节点、青萍温湿度/空气检测仪、执行器/LED |
| 协议/接口 | Zigbee/MQTT/VSOA可选 |
| 负责人 | 王道嘉 |
| 协作人 | 苏虹宇、方宏波 |
| 成果物 | 代码+指导书+PPT |

## 实验目标

基于温湿度/空气质量阈值触发告警、LED/蜂鸣器/消息推送联动

## 备注

面向课程展示的综合场景

## 目录结构


## 每周任务

| 周次 | 工作重点 | 主要任务 | 本周交付物 | 验收标准 |
| --- | --- | --- | --- | --- |
