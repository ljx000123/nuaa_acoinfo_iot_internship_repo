# 公共工程模板与测试框架

## 任务定位

统一 Git 目录规范、实验指导书模板、PPT 模板、测试记录表、日志规范和脚本化检查方式。

## 关联实验

支撑全部 16 个实验

## 目录结构

```text
B00_common_framework/
  src/      桥接或测试相关代码
  config/   topic、schema、服务地址等配置
  docs/     接口说明、测试说明、联调记录
  slides/   汇报材料
  tests/    自动化测试、验收用例
  assets/   日志截图、链路图、测试截图
```
