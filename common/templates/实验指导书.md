实验指导书字符设备驱动实验

V1.0.0

实验目的

了解字符设备驱动程序的基本结构和工作原理。

掌握 SylixOS系统中字符设备驱动的开发流程。

掌握字符设备驱动与用户空间应用程序的交互方法。

能够使用工具对字符设备驱动进行调试和验证。

实验内容

阅读 SylixOS 中字符设备驱动框架和接口说明，了解驱动程序的功能和实现方法。

编写字符设备驱动程序，完成设备的注册、文件操作接口的实现和设备文件的创建。

使用提供的接口注册字符设备驱动。

使用提供的接口将字符设备添加到系统设备管理链表。

实验环境

硬件：PC主机，弹性数字平台

软件：RealEvo-IDE、Putty

实验原理

设备驱动隐藏硬件的具体细节，并向其他部分提供一个抽象的、通用的接口。

在 SylixOS 中，设备驱动是内核空间的一部分，其运行在内核态下，驱动与底层硬件直接打交道，按照硬件设备的具体工作方式，读写设备的寄存器，完成设备的轮询、中断处理、DMA 通信，进行物理内存向虚拟内存的映射等，最终让通信设备能收发数据，让显示设备能显示文字和画面，让存储设备能记录文件和数据。

SylixOS 设备驱动是操作系统内核与硬件的接口，它把用户进程对于硬件设备的控制抽象为系统调用，并通过驱动程序提供的文件操作接口实现对实际硬件的特定操作，其关系如下图所示。

通过这种方法，应用程序可以像操作普通文件一样操作硬件设备，用户程序只需要关心这个抽象出来的文件，而一切同硬件打交道的工作都交给驱动程序来完成。

结合本实验内容，以下是SylixOS 设备管理分层模型

| 层次 | 代表组件 | 功能 |
| --- | --- | --- |
| 用户空间 (User Space) | Shell、应用进程 | 通过标准 POSIX 调用 open / read / write / ioctl 等访问设备节点(后续编写的测试程序位于此层) |
| 虚拟文件系统 (VFS) | devfs、Vnode | 将设备统一抽象为“文件”，解析路径并定位到驱动号与设备头(解析路径 /dev/ipc，定位到 _G_ipcDevHeader) |
| I/O System (IOS) | 驱动表、设备表 | 维护驱动号与回调函数；将 VFS 请求转发给对应驱动(iosDrvInstallEx、iosDevAdd 在此层完成注册) |
| 驱动层 (Driver) | file_operations 回调集合 | 直接操作硬件或内核服务，实现数据传输与控制(__ipcOpen/__ipcRead/__ipcWrite 等五大接口) |
| 硬件 / 内核服务 | 串口控制器、消息队列等 | 提供真正的数据读写能力(本实验以消息队列模拟硬件 FIFO) |

字符设备是指只能以字节为单位进行读写，读取数据需按照先后顺序，不能随机读取设备内存中某一数据的设备。常见的字符设备如：鼠标、键盘、串口等。在 SylixOS 中，每个字符设备都会在/dev 目录下对应一个设备文件，用户程序可通过设备文件（或设备节点）来进行字符设备的读写、IO 控制等操作。字符设备驱动也是驱动程序开发的基础内容之一，它为用户程序提供了与设备交互的接口。字符设备驱动的主要特性如下：

按字节流方式读写： 字符设备按字节流对数据进行读写，如串口设备或键盘。

灵活性： 通过文件操作接口实现对设备的精细控制。

SylixOS 中字符设备的驱动类型分为以下三类：

LW_DRV_TYPE_ORIG（原始设备驱动，兼容 VxWorks）。

LW_DRV_TYPE_SOCKET（SOCKET 型设备驱动）。

LW_DRV_TYPE_NEW_1（增强型设备驱动，支持文件权限管理和锁机制）。

字符设备驱动核心接口如下

| 接口 | 功能 | 代码位置 | 可选扩展 |
| --- | --- | --- | --- |
| open | 增减引用计数，分配私有资源 | __ipcOpen | 独占/共享模式、私有缓冲区 |
| close | 释放资源，引用计数减一 | __ipcClose | 计数清零时自动回收队列 |
| read | 复制数据到用户区，返回字节数或错误码 | __ipcRead | 非阻塞读、超时处理 |
| write | 从用户区取数据写入底层 | __ipcWrite | 写缓冲区满时挂起或丢包统计 |
| ioctl | 自定义控制命令 | __ipcIoctl | IPC_IOCTL_SET_MODE 调整传输模式 |

字符设备驱动的开发需要实现以下主要步骤：

驱动注册：

调用 API_IosDrvInstall、API_IosDrvInstallEx 或 API_IosDrvInstallEx2 注册驱动程序到内核驱动程序表中。

注册时需提供字符设备的基本操作函数（如 open、read、write 等）。

设备创建和管理：

使用 LW_DEV_HDR 定义设备头结构体并初始化相关成员。

调用 API_IosDevAdd 或 API_IosDevAddEx 将设备添加到设备管理链表中。

设备文件操作：

用户程序通过设备名访问设备，对应的驱动程序根据设备管理链表找到设备头，调用驱动函数完成读写和 IO 控制操作。

消息队列在本实验中的作用：

| 模式选择 | 处理方式 | 调用API | 教学目的 |
| --- | --- | --- | --- |
| TEST_MODE == 0（默认） | 使用 消息队列 模拟硬件 FIFO | API_MsgQueueSend/Receive | 让学生体验阻塞式驱动读写模型 |
| TEST_MODE == 1 | 使用 全局循环缓冲区 | memcpy | 对比同步/异步两种实现 |

static LW_DEV_HDR *_G_ipcDevHeader; /* ipc 设备驱动 */

#if TEST_MODE == 0

/*********************************************************************

** 函数名称: __ipcOpen

** 功能描述: 设备打开

** 输 入 : pdev 设备结构

** pcName 设备名

** iFlag 打开方式

** iMode 设备类型

** 输 出 : 字符设备

** 全局变量:

** 调用模块:

**************************************************************************/

static LONG __ipcOpen (LW_DEV_HDR *pdev, PCHAR pcName, INT iFlag, INT iMode)

{

LW_DEV_INC_USE_COUNT(pdev); /* 设备打开计数加一 */

return ((LONG)pdev);

}

/*************************************************************************

** 函数名称: __ipcClose

** 功能描述: 设备关闭

** 输 入 : pdev 设备结构

** 输 出 : ERROR CODE

** 全局变量:

** 调用模块:

**************************************************************************/

static INT __ipcClose (LW_DEV_HDR *pdev)

{

LW_DEV_DEC_USE_COUNT(pdev); /* 设备打开计数减一 */

return (ERROR_NONE);

}

/*******************************************************************

** 函数名称: __ipcRead

** 功能描述: 读功能

** 输 入 : pdev 设备驱动结构体

** cBuffer 读取的字符串

** stLength 数据长度

** 输 出 : 长度

** 全局变量:

** 调用模块:

************************************************************************/

static ssize_t __ipcRead (LW_DEV_HDR *pdev, CHAR *cBuffer, size_t stLength)

{

INT iError;

size_t stLen;

if (cBuffer == LW_NULL) { /* 指针有效性判断 */

_ErrorHandle(ERROR_IO_NO_DEVICE_NAME_IN_PATH);

return (PX_ERROR);

}

#if TEST_MODE == 0

iError = API_MsgQueueReceive(_G_MsgQueue, cBuffer, stLength, &stLen, LW_OPTION_WAIT_INFINITE);

if (iError != ERROR_NONE) {

printk("mesage queue receive fail.\n");

return (-1);

}

return (stLen);

#else

memcpy(cBuffer, _G_cBuffer, stLength);

bzero(_G_cBuffer, sizeof(_G_cBuffer));

return (ERROR_NONE);

#endif

}

** stLength 数据长度

** 输 出 : 长度

** 全局变量:

** 调用模块:

/*************************************************************************

** 函数名称: module_init

** 功能描述: 驱动注册

** 输 入 : NONE

** 输 出 : ERROR CODE

** 全局变量:

** 调用模块:

*************************************************************************/

INT module_init (VOID)

{

INT iRet;

struct file_operations fileop; /* 定义设备文件操作块 */

_G_ipcDevHeader = (LW_DEV_HDR*)__SHEAP_ZALLOC(sizeof(LW_DEV_HDR)); /* 设置内核堆空间 */

if (_G_ipcDevHeader == NULL) {

printk("memory low.\r\n");

return (PX_ERROR);

}

lib_memset(&fileop, 0, sizeof(struct file_operations)); /* 初始化设备文件操作块 */

/*

* 挂接驱动设备自定义函数

*/

fileop.fo_open = __ipcOpen; /* OPEN 函数实现 */

fileop.fo_close = __ipcClose; /* CLOSE 函数实现 */

fileop.fo_read = __ipcRead; /* READ 函数实现 */

fileop.fo_write = __ipcWrite; /* WRITE 函数实现 */

fileop.fo_ioctl = __ipcIoctl; /* IOCTL 函数实现 */

_G_iIpcDrvNum = iosDrvInstallEx(&fileop); /* 注册设备驱动 */

if (_G_iIpcDrvNum < 0) {

printk("driver install fail.\r\n");

__SHEAP_FREE(_G_ipcDevHeader);

return (PX_ERROR);

}

/*

* 驱动信息

*/

DRIVER_LICENSE(_G_iIpcDrvNum, "GPL->Ver 2.0");

DRIVER_AUTHOR(_G_iIpcDrvNum, "Wang.Kaixuan");

DRIVER_DESCRIPTION(_G_iIpcDrvNum, "ipc device driver.");

iRet = iosDevAdd(_G_ipcDevHeader, IPC_DEV_NAME, _G_iIpcDrvNum);

if (iRet < 0) {

printk("device add fail.\r\n");

iosDrvRemove(_G_iIpcDrvNum, TRUE);

__SHEAP_FREE(_G_ipcDevHeader);

return (PX_ERROR);

}

#if TEST_MODE == 0

_G_MsgQueue = API_MsgQueueCreate("irq_msgq",

MSG_COUNT,

BUFFER_SIZE,

LW_OPTION_WAIT_FIFO | LW_OPTION_OBJECT_LOCAL,

LW_NULL);

if (_G_MsgQueue == LW_OBJECT_HANDLE_INVALID) {

printk("mesage queue create fail.\n");

}

#endif

return (ERROR_NONE);

}

/* ASCII 网格参数 */

#define GRID_WIDTH 40 // 网格宽度（列数）

#define GRID_HEIGHT 40 // 网格高度（行数）

#define MAX_RANGE 6.0 // 显示范围（单位：米），超过此距离的点不显示

struct {

struct lidar data;

} lidar_server;

static sem_t sem;

static int get_lidar_data (int fd, struct lidar *lidar_data)

{

unsigned char data;

int retry = 10;

struct laser_data radar_data_tmp;

while (retry--) {

read(fd, &data, 1);

lidarDataGet(data);

if (dataInterface(&radar_data_tmp)) {

lidar_data->st1.num = radar_data_tmp.usPointsNum;

memcpy(lidar_data->st1.st2, radar_data_tmp.Points, sizeof(struct laserData) * 720);

return (1);

}

}

return (0);

}

/*

* lidar server

*/

static void *lidar_thread (void *arg)

{

int fd = *(int *)arg;

while (1) {

if (get_lidar_data(fd, &lidar_server.data)) {

sem_post(&sem);

}

}

return (NULL);

}

/*

* 主函数：打开串口设备、启动后台数据采集线程，并以字符画方式在终端输出点云数据

*/

int main(int argc, char **argv)

{

int fd;

pthread_t id;

struct timespec timeout = { 1, 0 };

fd = open(LIDAR_DEVICE, O_RDWR);

if (fd < 0) {

printf("open device error.\n");

return -1;

}

// 配置串口参数

ioctl(fd, SIO_BAUD_SET, 115200);

ioctl(fd, SIO_HW_OPTS_SET, CS8);

ioctl(fd, FIORBUFSET, 256);

sem_init(&sem, 0, 0);

// 创建后台激光雷达数据读取线程

pthread_create(&id, NULL, lidar_thread, (void *)(&fd));

while (1) {

// 等待数据帧到来（1秒超时）

if (sem_timedwait(&sem, &timeout) == 0) {

/* 构造 ASCII 网格 */

char grid[GRID_HEIGHT][GRID_WIDTH + 1];

int i;

int center_x = GRID_WIDTH / 2;

int center_y = GRID_HEIGHT / 2;

// 用空格填充网格，并添加行结束符

for (i = 0; i < GRID_HEIGHT; i++) {

memset(grid[i], ' ', GRID_WIDTH);

grid[i][GRID_WIDTH] = '\0';

}

// 将传感器位置标记为 'O'

grid[center_y][center_x] = 'O';

/* 计算缩放因子，使 MAX_RANGE 范围内的数据能够显示在网格内 */

float scale_x = ((float)(center_x - 1)) / MAX_RANGE;

float scale_y = ((float)(center_y - 1)) / MAX_RANGE;

float scale = (scale_x < scale_y) ? scale_x : scale_y;

/*

* 遍历所有采样点，计算它们在网格中的位置

* 假设点云的极坐标数据：

* - angle (单位：°) 为从传感器正右侧开始逆时针旋转的角度

* - distance 为距离（单位：米）

* 转换为笛卡尔坐标后，再映射到网格上

*/

for (i = 0; i < lidar_server.data.st1.num; i++) {

float angle_deg = lidar_server.data.st1.st2[i].angle;

float dist = lidar_server.data.st1.st2[i].distance;

// 如果距离超过显示范围，则跳过该点

if (dist > MAX_RANGE)

continue;

float rad = angle_deg * M_PI / 180.0;

int x = center_x + (int)(dist * cos(rad) * scale + 0.5);

int y = center_y - (int)(dist * sin(rad) * scale + 0.5); // y 轴反向

if (x >= 0 && x < GRID_WIDTH && y >= 0 && y < GRID_HEIGHT) {

grid[y][x] = '*';

}

}

// 使用 ANSI 转义码清屏，并将光标移至屏幕左上角

printf("\033[2J\033[H");

// 输出 ASCII 网格

for (i = 0; i < GRID_HEIGHT; i++) {

printf("%s\n", grid[i]);

}

// 输出附加信息（如采样点数量）

printf("采样点数量：%u\n", lidar_server.data.st1.num);

}

}

return 0;

}
