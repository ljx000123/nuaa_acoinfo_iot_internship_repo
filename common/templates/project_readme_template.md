## 6. 快速开始

### 6.1 获取代码

```bash
git clone https://github.com/ljx000123/nuaa_acoinfo_iot_internship_repo.git
cd nuaa_acoinfo_iot_internship_repo/[项目目录]
```

### 6.2 安装依赖

```bash
pip install -r requirements.txt
```

如果项目不是 Python 项目，请在此处填写对应的环境配置方式，例如 IDE 工程导入、SDK 安装、串口驱动安装或烧录工具配置。

### 6.3 修改配置

配置文件位置：

```text
config/
```

需要重点检查：

- 设备编号
- 串口号或网络地址
- MQTT broker 地址
- topic 规则
- 传感器参数
- 测试数据路径

### 6.4 运行示例

```bash
python scripts/demo.py
```

或填写项目实际运行命令。

## 10. 协作说明

- 每个实验单独建立目录，命名格式建议为 `[实验编号]_[实验简写]`。
- 新增代码放入 `src/` 或对应实验目录的 `src/`。
- 配置文件放入 `config/`。
- 测试脚本放入 `tests/`。
- 实验截图、流程图、结果图放入 `assets/`。
- 修改完成后建议通过 Pull Request 提交。
- 提交前请确认 README、测试记录和验收清单已同步更新。

## 11. 常见问题

| 问题 | 可能原因 | 解决方法 |
| --- | --- | --- |
| 环境无法运行 | 依赖未安装或版本不匹配 | 重新安装依赖并检查版本 |
