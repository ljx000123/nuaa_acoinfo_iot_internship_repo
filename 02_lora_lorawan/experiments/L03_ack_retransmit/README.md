# L3 LoRa可靠传输：ACK、重传与丢包统计

| 项目 | 内容 |
| --- | --- |
| 方向 | LoRa |
| 类型 | 进阶 |
| 难度 | 中 |

## 实验目标

实现序号、ACK、超时重传、丢包率统计，对比不同参数下的传输效果

## 备注

为综合场景提供可靠通信模块
