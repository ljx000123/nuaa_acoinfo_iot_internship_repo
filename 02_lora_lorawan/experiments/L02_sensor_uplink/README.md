# L2 LoRa传感数据采集与上报实验

## 基本信息

| 项目 | 内容 |
| --- | --- |
| 方向 | LoRa |
| 类型 | 入门 |
| 难度 | 低 |
| 主要设备 | ESP32-S3+SX126x、ESP32-C6显示板、传感器或模拟传感源 |
| 协议/接口 | LoRa P2P/JSON |
| 负责人 | 范文尧 |
| 协作人 | 熊依萱 |
| 成果物 | 代码+指导书+PPT |

## 实验目标

采集温湿度/空气质量或模拟数据，通过LoRa发送并在接收端显示/打印

## 备注

可先用模拟数据，后续替换真实传感源

## 目录结构

```text
L02_sensor_uplink/
  src/      实验源码
  config/   配置文件
  docs/     实验指导书、说明文档
  slides/   PPT 或汇报材料
  tests/    测试脚本、测试记录
  assets/   接线图、截图、素材
```

## 每周任务

| 周次 | 工作重点 | 主要任务 | 本周交付物 | 验收标准 |
| --- | --- | --- | --- | --- |
| 第1周 | LoRa P2P基础 | 清点ESP32-S3/SX126x与ESP32-C6显示板；搭建ESP32/Arduino或ESP-IDF环境；跑通LoRa点对点收发demo。 | L1/L2实验大纲、LoRa参数表、基础收发代码目录。 | 两块LoRa节点能稳定互发Hello包并输出RSSI/SNR。 |
| 第2周 | 完成LoRa基础与传感实验 | 完善L1点对点收发代码；实现L2传感/模拟数据封装为JSON并通过LoRa发送，接收端打印/显示。 | L1/L2代码、参数说明、指导书初稿、PPT提纲、运行截图。 | 两人互换设备后仍能按指导书复现L1/L2。 |
| 第3周 | 完善LoRa基础并支援上下行控制 | 修订L1/L2；将L2数据格式适配L5上行桥接；支援L6节点侧下行命令解析和执行。 | L1/L2二版、L6节点侧代码片段、指导书修订、PPT主体。 | L2数据能被桥接程序解析；L6节点能执行至少一种下行命令。 |
| 第4周 | LoRa基础课程定稿 | 根据复现意见修订L1/L2；补充LoRa参数解释、截图、思考题；支援L8演示。 | L1/L2最终代码、指导书定稿、PPT定稿、复现记录。 | L1/L2可作为后续LoRa实验统一模板。 |
